#  Java Quiz Application

A simple console-based quiz system built using Java and OOP concepts.

---

##  Features
- Multiple-choice questions (MCQ)
- Instant answer checking
- Automatic score & percentage
- Grade system (PASS/FAIL)
- Case-insensitive input

---

## Project Structure
Main.java          # Entry point  
SimpleQuiz.java    # Quiz logic  
Question.java      # Question data  

---

##  How to Run
javac Main.java  
java Main  

---

##  Sample Output
 QUIZ STARTED!
Total: 30 marks

1. 2+2=?
A) 3  B) 4  C) 5  D) 6

 Correct!
